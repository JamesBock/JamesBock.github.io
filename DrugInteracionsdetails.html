Drug Interaction Module

This project started as as experiment with learning Blazor and grew into a practical tool that can be used in clinical and consumer settings.

The module is designed with the use-case of an e-health record pinging the module with a patient ID that can be used to retrive the medications the patient is taking. The module then gathers the proper data, converts it into a readable form for the National Library of Medicine's Drug Interaction API, and parses the information returned from the API. Once the interactions are parsed, they must be connected with the order the prescriber wrote so modifications can be made if necessary based on the interactions.

As shown above, the interface is adaptive so it can be accessed on mobile devices if desired.
