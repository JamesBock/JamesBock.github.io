DrugInteractionAPI

Based on the custom parsing from my Drug Interaction Module, I expanded the functionalliy and created a library for modular use.

This package adds functionallity for the single medication endpoint of the drug interaction API, which returns a response of all the drugs that assert an interaction with the medication sent to the endpoint. The parser in the Drug Interaction Module only needed support for the list medication endpoint that finds interactions between members of the list.

Repository includes integration tests that verify proper access to the endpoints and that parsing is working for single and multi- drug medications.
