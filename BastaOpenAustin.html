    Tenant File Web App

    A web application to help tenants keep track of documents related to their rental unit

    Tenants, especially low-income tenants, and advocates need to be safe and organize records related to units and complexes. Records include photos, receipts, the lease terms, notes from the landlord, fees/fines, and incident reports. We're looking for an accessible platform designed for those that BASTA works with -- easy to text or upload in, making metadata clear, and with flexible security settings.

    I contributed primarily on the backend of the project, building the graphQL server with dataloaders and integrating a number of cloud assets from Google Cloud Platform.

    With these assets implemented, the app can assess images sent via MMS and label them with automated descriptions. If text is detected, Occular Character Recognition is used to scan the document for keywords and phrases that are used to direct the text conversation flow that the tenant is having with our Twilio bot.

    Automated content moderation is also used to reject or sequester images that are inappropriate or privacy concerns.

    The primary image above exhibits many of these capabilities. It shows the graphQL playground querying the database for images of a specific tenant. The labels associated with a specific image are shown. Text was found in the particular image and OCR picked up a keypharse that was persisted to the database.
